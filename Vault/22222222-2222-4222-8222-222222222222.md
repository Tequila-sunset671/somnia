# Somnia 🌙

A native macOS browser fused with an Obsidian-style notes vault. Browse the web
on the left, keep linked markdown notes on the right — and see them as a graph.
Built with SwiftUI + WebKit. No Electron, zero external dependencies.

> This note mirrors the project README (see [[Welcome to Somnia]]).

## Why Somnia?

Zen and Orion are browsers; Obsidian is notes. Somnia is both, in one native app
— a browser whose second half is a real linked-notes vault with a graph.

## Features

Browser: spaces (tab groups); lazy tabs that sleep on an LRU budget and release
memory under pressure; find on page (⌘F); history; address autocomplete;
favicons; downloads manager; configurable search engine; Reader Mode (⌘⇧R).

Notes: markdown vault with [[wiki-links]] and backlinks; force-directed graph;
connect an external Obsidian vault (read-only).

Privacy: native tracker/ad blocking; private tabs (⌘⇧N); local-first data.

## Shortcuts

- ⌘T new tab · ⌘⇧N private tab · ⌘W close · ⌘1–9 jump
- ⌘K quick open · ⌘F find · ⌘⇧R reader · ⌘L location · ⌘, customize
- ⌘R reload · ⌘[ / ⌘] back / forward

## Build

Requires macOS 14+ and Xcode Command Line Tools (not full Xcode).

```
./build.sh      # builds Somnia.app with swiftc
open Somnia.app
./test.sh       # pure-logic unit tests
```

The app is ad-hoc signed — right-click → Open on first launch, or build it
yourself.
